# Icon Setup

The extension needs three PNG icons in this directory:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

## Quick Icon Creation Options:

### Option 1: Use an Emoji (Easiest)
1. Go to https://favicon.io/emoji-favicons/card-file-box/
2. Download the favicon package
3. Rename the files to match the sizes above

### Option 2: Create Custom Icons
1. Use Figma, Canva, or any design tool
2. Create a 128x128 design with your logo
3. Export as PNG at 128x128, 48x48, and 16x16
